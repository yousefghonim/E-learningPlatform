Description:
This is a full-featured Hotel Management System built with Django and SQLite. It provides a complete web interface to manage hotel operations including room management, guest profiles, image galleries, and online payments through Razorpay.

Features:

User authentication (Admin, Staff, Guests)

CRUD operations for rooms, guest profiles, and bookings

Image upload and gallery management (requires Pillow)

Integration with Razorpay for online payment processing

Responsive front-end using Django templates

Media and static file management

Technologies Used:

Python 3.11

Django 4.x

SQLite (default database)

Razorpay API for payments

Pillow for image handling

HTML, CSS, and JavaScript for front-end
